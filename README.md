# pokemon

簡單的前端範例網站（純 HTML / CSS / JS，無框架）。

內容：

- `index.html`：主要頁面
- `styles.css`：樣式表
- `app.js`：互動腳本（導覽切換、表單驗證）

快速使用：

1. 在專案根目錄啟動靜態伺服器，例如使用 Python 3：

```
python3 -m http.server 8000
```

2. 在瀏覽器開啟 http://localhost:8000

如需進一步自訂或加入資源圖示、圖片，請在專案中建立 `assets/` 資料夾並更新 `index.html`。
